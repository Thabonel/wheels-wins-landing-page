How to create your Open Graph image:

  1. Open this file in Chrome or Firefox
  2. Set browser zoom to 100%
  3. Use a screenshot tool to capture ONLY the blue area (1200x630px)
  4. Save as "opengraph-image.png" in the /public folder
  5. Delete this HTML file after creating the image

Pro tip: Use Chrome DevTools (F12) → Device Toolbar → Set custom size to 1200x630px for perfect sizing
